## 每日程式撰寫問題: 問題 #187 [簡單]

早安！ 以下是今日給你的程式撰寫面試問題。

這個問題是由 Google 所問。

給定一列矩型以最小以及最大的 x 軸與 y 軸。計算是否兩個矩型會相互覆蓋。如果有一個矩型完全覆蓋了另一個矩型，這就是所謂覆蓋。

舉例來說，給定以下矩型：

    {
        "top_left": (1, 4),
        "dimensions": (3, 3) # 寬, 高
    },
    {
        "top_left": (-1, 3),
        "dimensions": (2, 1)
    },
    {
        "top_left": (0, 5),
        "dimensions": (4, 3)
    }

回傳 true，因為第一個與第三個矩型相互覆蓋。
