## 每日程式撰寫問題: 問題 #68 [中等]

早安！ 以下是今日給你的程式撰寫面試問題。

這個問題是由 Google 所問。

在一個特製的西洋棋盤，兩個主教會相互攻擊如果彼此在斜角方向有連線到。這包含其兩個主教連線中有一個主教的狀況。也就是主教可以穿過棋子攻擊。

給定 N 個主教，以 (列, 行) 這來表示在 M 列 M 行的西洋棋盤。寫一個函式來計算多少對主教可以彼此攻擊。配對的順序不計，也就是配對 (1, 2) 與配對 (2, 1) 當作同一組。

舉例，給定 M = 5 以及一列主教：

- (0, 0)
- (1, 2)
- (2, 2)
- (4, 0)


棋盤將會如下：

    [b 0 0 0 0]
    [0 0 b 0 0]
    [0 0 b 0 0]
    [0 0 0 0 0]
    [b 0 0 0 0]

則會回傳 2 ， 因為主教 1 與主教 3 會相互攻擊，相同地，主教 3 與主教 4 也會。
